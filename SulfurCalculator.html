About This Tool

Purpose

This calibration curve calculator was developed specifically for the correction of sulfur quantification using the Sindie MWDXRF (Monochromatic Wavelength Dispersive X-Ray Fluorescence) analyzer.

The tool implements the ASTM D7309 standard method for determining sulfur content in petroleum products and lubricants, providing accurate calibration curves with daily correction factors.

Key Features

  • Dual regression analysis (Linear & Log-Log)
  • Independent point selection for each curve type
  • Daily correction factor calculation
  • Real-time sample analysis with CF application
  • Interactive calibration detail viewer with charts
  • PDF report generation for sample results
  • Local browser storage for calibration data
  • ASTM D7309 compliant calculations

Workflow

  • Option A: Use existing saved calibration (no new calibration needed)
  • Option B: Create new calibration with standards and save it
  • Step 1: Activate a saved calibration
  • Step 2: Run reference standard to calculate daily correction factor
  • Step 3: Analyze samples with automatic CF application
  • Step 4: Print PDF report or export results
  • View Details: Click 👁️ View button to see calibration curves and data points
